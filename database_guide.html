Metode 1: Via phpMyAdmin (Recommended)

  1. Buka phpMyAdmin
  2. Klik menu "Databases"
  3. Buat database baru: peminjaman_barang_db
  4. Klik database yang baru dibuat
  5. Klik tab "Import"
  6. Klik "Choose File" dan pilih database/peminjaman_barang_simple.sql
  7. Klik "Go"

Metode 2: Via Command Line

mysql -u root -p peminjaman_barang_db < database/peminjaman_barang_simple.sql

Untuk Windows PowerShell:

Get-Content database\peminjaman_barang_simple.sql | mysql -u root -p peminjaman_barang_db

Kunjungi: http://localhost/web_sunsal/test_import.php

  • Test koneksi ke semua tabel

Jika Import Gagal:

  1. Hapus database lama: DROP DATABASE peminjaman_barang_db;
  2. Buat ulang database: CREATE DATABASE peminjaman_barang_db;
  3. Gunakan file simple: peminjaman_barang_simple.sql
  4. Nonaktifkan foreign key: SET FOREIGN_KEY_CHECKS = 0;
